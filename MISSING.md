## Contributing

If you have any of these downloaded, please either [contact me](https://yello.ooo), submit an issue, or a pull request.

You should only send `update.lst` or `setup.lst` files, as well as the filename of the downloaded zip, if applicable.

The easiest way for you to contribute is to run the `7z x -o"lists" -r *.zip *.lst`
command from inside the folder you have your downloaded update and DLC .zip files,
and then zip up the created `lists` folder and send that to me.
