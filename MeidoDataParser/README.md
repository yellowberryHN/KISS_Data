# MeidoDataParser 

A tool to parse KISS update.lst files and export them as a KISS_Data .json file.

## Attributions

Fetches data from the following sources to check for missing DLCs in the dataset:

- [MeidosFriend/CM3D2_DLC_Checker](https://github.com/MeidosFriend/CM3D2_DLC_Checker)
- [krypto5863/COM3D2_DLC_Checker](https://github.com/krypto5863/COM3D2_DLC_Checker)
- [MeidosFriend/COM3D2_EN_DLC_Checker](https://github.com/MeidosFriend/COM3D2_EN_DLC_Checker)
- [MeidosFriend/COM3D2_INM_DLC_Checker](https://github.com/MeidosFriend/COM3D2_INM_DLC_Checker)
- [MeidosFriend/CRE_DLC_Checker](https://github.com/MeidosFriend/CRE_DLC_Checker)

Thank you for your hard work! This project would be a lot harder without these sources.
